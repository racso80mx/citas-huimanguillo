# **App Name**: CitaMedicaFacil

## Core Features:

- User Data Input: Collect user's CURP, Names, Last Names, Sex, and Age for appointment booking. Simplify input by pre-filling some fields from CURP where possible.
- Appointment Date Selection: Allow users to select the desired appointment date with a clear calendar interface.
- Availability Check: Check appointment availability for the selected date and display remaining slots.
- Real-time Availability Indicator: Show a real-time indicator of available slots and disable fully booked dates in the calendar.
- Appointment Booking: Enable users to book appointments once availability is confirmed.
- Admin Appointment Report: Provide administrators access to reports of booked appointments.
- Appointment Confirmation: Send an immediate confirmation to users who succesfully schedule their appointment

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) to evoke calmness and trust.
- Background color: Very light blue (#F0F8FF) to ensure a clean and accessible layout.
- Accent color: Soft green (#90EE90) to highlight available slots and important actions.
- Body and headline font: 'PT Sans' for readability and a modern yet warm feel.
- Simple, clean layout with a focus on larger, easily tappable buttons for users with limited tech experience.
- Use clear, universal icons to represent appointment types and actions.
- Subtle animations to provide feedback on actions, such as confirming a booking.
